# 💰 Personal Finance Tracker

Aplikasi pencatat keuangan pribadi berbasis web yang membantu pengguna melacak pemasukan dan pengeluaran secara real-time dengan visualisasi data yang interaktif.

## 🚀 Live Demo

Cek aplikasinya di sini: (https://wijdannuramzad.github.io/personal-finance-tracker/)

## ✨ Fitur Utama

- **CRUD Transaksi:** Tambah dan hapus data pemasukan/pengeluaran.
- **Local Storage:** Data tersimpan aman di browser, tidak hilang saat refresh.
- **Data Visualization:** Grafik lingkaran (Doughnut Chart) menggunakan **Chart.js**.
- **Auto Calculation:** Perhitungan otomatis sisa saldo, total pemasukan, dan total pengeluaran.

## 🛠️ Tech Stack

- HTML5 & CSS3 (Minimalist White UI)
- Vanilla JavaScript
- Chart.js (External Library via CDN)
